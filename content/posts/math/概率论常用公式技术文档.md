# 概率论常用公式技术文档

**版本**：1.0  
**最后更新**：2026年5月  
**适用范围**：工程、统计学、机器学习、数据分析等领域的概率计算

---

## 1. 符号与基本公理

- 样本空间 \(\Omega\)，事件 \(A, B, C, \ldots\)
- \(P(A)\)：事件 \(A\) 的概率
- 公理：
  1. \(0 \le P(A) \le 1\)
  2. \(P(\Omega) = 1\)
  3. 若 \(A_1, A_2, \ldots\) 互斥，则 \(P(\bigcup_{i=1}^\infty A_i) = \sum_{i=1}^\infty P(A_i)\)

## 2. 概率的基本性质

| 公式 | 说明 |
|------|------|
| \(P(\emptyset) = 0\) | 不可能事件概率为0 |
| \(P(\bar{A}) = 1 - P(A)\) | 对立事件 |
| \(P(A \setminus B) = P(A) - P(A \cap B)\) | 差事件 |
| \(P(A \cup B) = P(A) + P(B) - P(A \cap B)\) | 加法公式（两事件） |
| \(P(\bigcup_{i=1}^n A_i) = \sum_{i} P(A_i) - \sum_{i<j} P(A_i \cap A_j) + \cdots + (-1)^{n-1} P(\bigcap_{i=1}^n A_i)\) | 容斥原理（多事件加法公式） |
| 若 \(A \subseteq B\)，则 \(P(A) \le P(B)\) | 单调性 |

## 3. 条件概率与独立性

### 3.1 条件概率
\[
P(A \mid B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0
\]

### 3.2 乘法公式与链式法则
- 两事件：\(P(A \cap B) = P(A)P(B \mid A) = P(B)P(A \mid B)\)
- **链式法则（n 个事件）**：  
  \[
  P(A_1 \cap A_2 \cap \cdots \cap A_n) = P(A_1) \prod_{i=2}^{n} P(A_i \mid A_1 \cap \cdots \cap A_{i-1})
  \]  
  前提：所有条件概率有定义。

### 3.3 全概率公式
设 \(\{B_i\}\) 是 \(\Omega\) 的一个划分（互斥且 \(\bigcup_i B_i = \Omega\)，\(P(B_i) > 0\)），则
\[
P(A) = \sum_i P(B_i) P(A \mid B_i)
\]

### 3.4 贝叶斯公式
\[
P(B_j \mid A) = \frac{P(B_j) P(A \mid B_j)}{\sum_i P(B_i) P(A \mid B_i)}
\]

### 3.5 条件化的推广（给定条件 C）
- 条件链式法则：  
  \(P(A_1 \cap \cdots \cap A_n \mid C) = \prod_{i=1}^n P(A_i \mid A_1 \cap \cdots \cap A_{i-1} \cap C)\)
- 条件全概率：  
  \(P(A \mid C) = \sum_i P(B_i \mid C) P(A \mid B_i \cap C)\)
- 条件贝叶斯：  
  \(P(B_j \mid A \cap C) = \frac{P(B_j \mid C) P(A \mid B_j \cap C)}{\sum_i P(B_i \mid C) P(A \mid B_i \cap C)}\)

### 3.6 独立性
- 两事件独立：\(P(A \cap B) = P(A)P(B)\)
- 多个事件相互独立：对任意子集，乘积分解成立。

## 4. 随机变量与分布

### 4.1 分布函数与密度
- 分布函数：\(F_X(x) = P(X \le x)\)
- 离散型：概率质量函数 \(p(x) = P(X=x)\)
- 连续型：概率密度函数 \(f(x)\)，满足 \(P(a \le X \le b) = \int_a^b f(x) dx\)，\(F'(x)=f(x)\)

### 4.2 常见离散分布

| 分布 | 参数 | \(P(X=k)\) | 期望 \(E[X]\) | 方差 \(\mathrm{Var}(X)\) |
|------|------|------------|---------------|--------------------------|
| 0-1分布 | \(p\in[0,1]\) | \(p^k(1-p)^{1-k}, k=0,1\) | \(p\) | \(p(1-p)\) |
| 二项分布 \(B(n,p)\) | \(n\ge1, 0\le p\le1\) | \(\binom{n}{k}p^k(1-p)^{n-k}, k=0..n\) | \(np\) | \(np(1-p)\) |
| 泊松分布 \(P(\lambda)\) | \(\lambda>0\) | \(\frac{e^{-\lambda}\lambda^k}{k!}, k=0,1,\ldots\) | \(\lambda\) | \(\lambda\) |
| 几何分布（首次成功） | \(p\in(0,1]\) | \((1-p)^{k-1}p, k=1,2,\ldots\) | \(1/p\) | \((1-p)/p^2\) |
| 超几何分布 | \(N, K, n\) | \(\frac{\binom{K}{k}\binom{N-K}{n-k}}{\binom{N}{n}}\) | \(n\frac{K}{N}\) | \(n\frac{K}{N}(1-\frac{K}{N})\frac{N-n}{N-1}\) |

### 4.3 常见连续分布

| 分布 | 参数 | 密度 \(f(x)\) | 期望 | 方差 |
|------|------|---------------|------|------|
| 均匀分布 \(U(a,b)\) | \(a<b\) | \(\frac{1}{b-a}, x\in[a,b]\) | \(\frac{a+b}{2}\) | \(\frac{(b-a)^2}{12}\) |
| 指数分布 \(\mathrm{Exp}(\lambda)\) | \(\lambda>0\) | \(\lambda e^{-\lambda x}, x\ge0\) | \(1/\lambda\) | \(1/\lambda^2\) |
| 正态分布 \(N(\mu,\sigma^2)\) | \(\mu\in\mathbb{R}, \sigma>0\) | \(\frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}\) | \(\mu\) | \(\sigma^2\) |
| 伽马分布 \(\Gamma(\alpha,\beta)\) | \(\alpha>0,\beta>0\) | \(\frac{\beta^\alpha}{\Gamma(\alpha)}x^{\alpha-1}e^{-\beta x}, x>0\) | \(\alpha/\beta\) | \(\alpha/\beta^2\) |
| 贝塔分布 \(\mathrm{Beta}(\alpha,\beta)\) | \(\alpha,\beta>0\) | \(\frac{x^{\alpha-1}(1-x)^{\beta-1}}{B(\alpha,\beta)}, x\in[0,1]\) | \(\frac{\alpha}{\alpha+\beta}\) | \(\frac{\alpha\beta}{(\alpha+\beta)^2(\alpha+\beta+1)}\) |

## 5. 数字特征与矩

### 5.1 期望
- 离散：\(E[X] = \sum_i x_i p_i\)
- 连续：\(E[X] = \int_{-\infty}^{\infty} x f(x) dx\)
- 线性：\(E[aX+bY+c] = aE[X]+bE[Y]+c\)
- 若 \(X,Y\) 独立：\(E[XY] = E[X]E[Y]\)

### 5.2 方差
- 定义：\(\mathrm{Var}(X) = E[(X-E[X])^2]\)
- 计算式：\(\mathrm{Var}(X) = E[X^2] - (E[X])^2\)
- 性质：\(\mathrm{Var}(aX+b) = a^2 \mathrm{Var}(X)\)
- 和方差：\(\mathrm{Var}(X\pm Y) = \mathrm{Var}(X) + \mathrm{Var}(Y) \pm 2\mathrm{Cov}(X,Y)\)
- 若 \(X,Y\) 独立，则 \(\mathrm{Var}(X\pm Y) = \mathrm{Var}(X) + \mathrm{Var}(Y)\)

### 5.3 协方差与相关系数
- 协方差：\(\mathrm{Cov}(X,Y) = E[(X-E[X])(Y-E[Y])] = E[XY] - E[X]E[Y]\)
- 性质：\(\mathrm{Cov}(X,X) = \mathrm{Var}(X)\)，对称双线性
- 相关系数：\(\rho_{XY} = \frac{\mathrm{Cov}(X,Y)}{\sqrt{\mathrm{Var}(X)\mathrm{Var}(Y)}} \in [-1,1]\)

### 5.4 高阶矩与矩母函数
- \(k\) 阶原点矩：\(\mu_k' = E[X^k]\)
- \(k\) 阶中心矩：\(\mu_k = E[(X-E[X])^k]\)
- 矩母函数：\(M_X(t) = E[e^{tX}]\)，满足 \(E[X^n] = M_X^{(n)}(0)\)（若存在）
- 特征函数：\(\varphi_X(t) = E[e^{itX}]\)，始终存在且唯一确定分布。

### 5.5 条件期望与方差
- 条件期望 \(E[X \mid Y]\) 是 \(Y\) 的函数。
- **全期望公式（Law of Total Expectation）**：  
  \(E[X] = E[E[X \mid Y]]\)
- **全方差公式（Law of Total Variance）**：  
  \(\mathrm{Var}(X) = E[\mathrm{Var}(X \mid Y)] + \mathrm{Var}(E[X \mid Y])\)
- 全协方差公式：  
  \(\mathrm{Cov}(X,Y) = E[\mathrm{Cov}(X,Y \mid Z)] + \mathrm{Cov}(E[X \mid Z], E[Y \mid Z])\)

## 6. 极限定理与不等式

### 6.1 切比雪夫不等式
\[
P(|X-\mu| \ge \varepsilon) \le \frac{\sigma^2}{\varepsilon^2}, \quad \mu = E[X], \sigma^2 = \mathrm{Var}(X)
\]
等价形式：\(P(|X-\mu| < k\sigma) \ge 1 - \frac{1}{k^2}\)

### 6.2 马尔可夫不等式（非负随机变量）
\[
P(X \ge a) \le \frac{E[X]}{a}, \quad a>0
\]

### 6.3 大数定律
- 弱大数定律：\(\bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{P} \mu\)
- 强大数定律：\(\bar{X}_n \xrightarrow{a.s.} \mu\)

### 6.4 中心极限定理
设 \(X_1,\ldots,X_n\) 独立同分布，\(E[X_i]=\mu\)，\(\mathrm{Var}(X_i)=\sigma^2\in(0,\infty)\)，则
\[
\frac{\sqrt{n}(\bar{X}_n - \mu)}{\sigma} \xrightarrow{d} N(0,1)
\]
即对充分大的 \(n\)，\(\sum_{i=1}^n X_i\) 近似服从 \(N(n\mu, n\sigma^2)\)。

### 6.5 De Moivre–Laplace 定理（二项分布逼近正态）
若 \(X \sim B(n,p)\)，当 \(n\) 大且 \(np>5, n(1-p)>5\) 时，
\[
P(X \le k) \approx \Phi\left( \frac{k+0.5 - np}{\sqrt{np(1-p)}} \right)
\]
其中 \(\Phi\) 是标准正态分布函数，\(+0.5\) 为连续性校正。

## 7. 其他常用公式

### 7.1 詹森不等式
若 \(\varphi\) 是凸函数，则 \(\varphi(E[X]) \le E[\varphi(X)]\)；若 \(\varphi\) 凹，则反向。

### 7.2 柯西-施瓦茨不等式
\[
(E[XY])^2 \le E[X^2] E[Y^2]
\]
等号成立当且仅当 \(X\) 与 \(Y\) 几乎必然成比例。

### 7.3 洛必达法则与概率极限（略）

### 7.4 随机变量函数的期望
- 离散：\(E[g(X)] = \sum_i g(x_i) p_i\)
- 连续：\(E[g(X)] = \int_{-\infty}^{\infty} g(x) f_X(x) dx\)
- 多个变量：\(E[g(X,Y)] = \iint g(x,y) f_{X,Y}(x,y) dxdy\)

### 7.5 卷积公式（独立随机变量和）
若 \(X,Y\) 独立，\(Z=X+Y\)，则
- 离散：\(p_Z(z) = \sum_x p_X(x) p_Y(z-x)\)
- 连续：\(f_Z(z) = \int_{-\infty}^{\infty} f_X(x) f_Y(z-x) dx\)

## 8. 常用技巧与注意事项

- **全概率 + 贝叶斯**：用于复杂事件的分情形推理。
- **链式法则**：处理多个依赖事件联合概率。
- **全期望公式**：分层计算期望，常用于随机过程。
- **独立性判断**：验证乘积分解或条件概率是否等于无条件。
- **正态分布标准化**：\(Z = \frac{X-\mu}{\sigma} \sim N(0,1)\)，查表使用。
