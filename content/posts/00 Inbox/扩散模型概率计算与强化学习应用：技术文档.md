## 第一章：扩散模型概率计算核心原理

### 1.1 模块边界与知识框架

本章涵盖DDPM（Denoising Diffusion Probabilistic Model）的核心概率计算体系，包括：前向扩散过程、反向去噪过程、变分下界优化、参数化技巧与训练目标、概率流与连续时间扩展。

**文档章节关系图**：

```
概率基础（1.1-1.2）→ 前向过程（1.3）→ 反向过程（1.4）→ 优化目标（1.5）→ 引导生成（1.6）→ 连续扩展（1.7）
```

### 1.2 关键概念：何为扩散模型？

**是什么**：扩散模型是一种通过**渐进式噪声注入**与**逐步去噪**来学习目标数据分布的概率生成模型。其核心假设是：任何复杂数据分布都可以通过马尔可夫链转化为简单的高斯分布，再通过可学习的逆向过程恢复原始数据。

**为什么需要**：
- 相比GAN，扩散模型**训练更稳定**，避免模式崩塌
- 相比VAE，生成**图像质量更高**，细节更丰富
- 相比自回归模型，**并行采样**能力更强

**解决什么问题**：高质量、高保真度的数据生成（图像、视频、音频、3D模型等），以及复杂分布的精确建模。

### 1.3 前向扩散过程（Forward Diffusion）

**定义与数学公式**：前向扩散过程通过T步逐步向原始数据 \(x_0\) 添加高斯噪声，每一步满足：

\[
q(x_t | x_{t-1}) = \mathcal{N}(x_t; \sqrt{1-\beta_t} \, x_{t-1}, \beta_t \mathbf{I})
\]

其中 \(\beta_t \in (0,1)\) 为预设的噪声调度系数，通常随时间步长逐渐增大。

**重参数化技巧（Reparameterization Trick）** ：利用高斯分布的可加性，可将 \(x_t\) 直接表示为 \(x_0\) 的闭式解：

\[
q(x_t | x_0) = \mathcal{N}(x_t; \sqrt{\bar{\alpha}_t} x_0, (1-\bar{\alpha}_t) \mathbf{I})
\]

其中 \(\alpha_t = 1 - \beta_t\)，\(\bar{\alpha}_t = \prod_{i=1}^{t} \alpha_i\)。

**核心作用**：这一设计使得前向过程无需迭代计算T步，训练时可直接通过随机采样时间步t \( \sim \mathcal{U}(1,T) \) 和噪声 \(\epsilon \sim \mathcal{N}(0, \mathbf{I})\) 生成任意时间步的噪声数据，极大提升了训练效率。

### 1.4 反向去噪过程（Reverse Diffusion）

**定义**：反向过程的目标是从纯噪声 \(x_T \sim \mathcal{N}(0, \mathbf{I})\) 出发，逐步去噪还原出原始数据 \(x_0\)。该过程同样定义为马尔可夫链：

\[
p_\theta(x_{t-1} | x_t) = \mathcal{N}(x_{t-1}; \mu_\theta(x_t, t), \Sigma_\theta(x_t, t))
\]

其中 \(\mu_\theta\) 和 \(\Sigma_\theta\) 由神经网络（如U-Net）预测。

**简化策略**：DDPM将 \(\Sigma_\theta\) 固定为常数，仅学习均值 \(\mu_\theta\)，并通过重参数化转化为噪声预测网络 \(\epsilon_\theta\)：

\[
\mu_\theta(x_t, t) = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1-\bar{\alpha}_t}} \epsilon_\theta(x_t, t) \right)
\]

**PyTorch实现示例**（核心结构）：

```python
import torch
import torch.nn as nn

class DiffusionModel(nn.Module):
    def __init__(self, unet, betas, timesteps=1000):
        super().__init__()
        self.unet = unet
        self.timesteps = timesteps
        # 预计算扩散系数
        self.betas = betas  # [0.0001, 0.0002, ..., 0.002]
        self.alphas = 1.0 - betas
        self.alpha_bars = torch.cumprod(self.alphas, dim=0)
        self.sqrt_alpha_bars = torch.sqrt(self.alpha_bars)
        self.sqrt_one_minus_alpha_bars = torch.sqrt(1.0 - self.alpha_bars)
    
    def q_sample(self, x0, t, noise=None):
        """前向过程：一步到位计算x_t"""
        if noise is None:
            noise = torch.randn_like(x0)
        # x_t = sqrt(alpha_bar) * x0 + sqrt(1-alpha_bar) * noise
        x_t = self.sqrt_alpha_bars[t][:, None, None, None] * x0 + \
              self.sqrt_one_minus_alpha_bars[t][:, None, None, None] * noise
        return x_t
    
    def p_sample(self, x_t, t):
        """反向过程一步：预测x_{t-1}"""
        # 预测噪声
        noise_pred = self.unet(x_t, t)
        # 计算x_{t-1}均值
        alpha_t = self.alphas[t]
        alpha_bar_t = self.alpha_bars[t]
        sqrt_alpha_t = torch.sqrt(alpha_t)
        beta_t = self.betas[t]
        sqrt_one_minus_alpha_bar_t = torch.sqrt(1 - alpha_bar_t)
        
        mu_theta = (1 / sqrt_alpha_t) * (x_t - beta_t / sqrt_one_minus_alpha_bar_t * noise_pred)
        
        # 采样x_{t-1}
        if t > 0:
            sigma_t = torch.sqrt(beta_t)
            z = torch.randn_like(x_t)
            return mu_theta + sigma_t * z
        return mu_theta
    
    def loss(self, x0, t, noise):
        """训练损失：预测噪声与真实噪声的MSE"""
        x_t = self.q_sample(x0, t, noise)
        noise_pred = self.unet(x_t, t)
        return nn.functional.mse_loss(noise_pred, noise)
```

### 1.5 优化目标：变分下界（ELBO）

**数学推导**：扩散模型的训练目标是对数似然的最大化。由于直接计算 \(\log p_\theta(x_0)\) 难以处理，转而最大化其证据下界：

\[
\log p_\theta(x_0) \geq \mathbb{E}_{q(x_{1:T}|x_0)} \left[ \log \frac{p_\theta(x_{0:T})}{q(x_{1:T}|x_0)} \right] \triangleq \text{ELBO}
\]

ELBO可以分解为三项之和：

\[
\text{ELBO} = \underbrace{\mathbb{E}_{q(x_1|x_0)}[\log p_\theta(x_0|x_1)]}_{\text{重构项}} 
- \underbrace{D_{\text{KL}}(q(x_T|x_0) \| p(x_T))}_{\text{先验匹配项}}
- \sum_{t=2}^T \underbrace{\mathbb{E}_{q(x_t|x_0)}[D_{\text{KL}}(q(x_{t-1}|x_t,x_0) \| p_\theta(x_{t-1}|x_t))]}_{\text{去噪匹配项}}
\]

**简化的训练目标**：DDPM证明了上述KL项可简化为简单的均方误差损失：

\[
\mathcal{L}_{\text{simple}} = \mathbb{E}_{t, x_0, \epsilon} \left[ \| \epsilon - \epsilon_\theta(\sqrt{\bar{\alpha}_t}x_0 + \sqrt{1-\bar{\alpha}_t}\epsilon, t) \|^2 \right]
\]

即让神经网络预测出原始噪声 \(\epsilon\)。

**训练流程**：
```
Algorithm 1: DDPM Training
Input: 训练数据集 D，总步数 T
For each training iteration:
  1. 采样数据样本 x0 ~ D
  2. 采样时间步 t ~ Uniform(1, T)
  3. 采样噪声 ε ~ N(0, I)
  4. 计算 x_t = sqrt(ᾱ_t)·x0 + sqrt(1-ᾱ_t)·ε
  5. 预测噪声 ε̂ = εθ(x_t, t)
  6. 计算损失 L = MSE(ε, ε̂)
  7. 反向传播更新 εθ
```

### 1.6 条件生成与引导方法

**为什么需要引导**：无条件扩散模型生成的数据分布是训练数据的精确建模。若需控制生成结果（如“生成一只猫”），则需要将条件信息 \(y\) 融入生成过程。

#### 1.6.1 分类器引导（Classifier Guidance, CG）

在推理阶段，利用外部分类器 \(p_\phi(y|x_t)\) 的条件梯度修正采样方向：

\[
\nabla_{x_t} \log p_\theta(x_t|y) \approx \nabla_{x_t} \log p_\theta(x_t) + \gamma \nabla_{x_t} \log p_\phi(y|x_t)
\]

其中 \(\gamma\) 为引导强度。此方法需要额外训练分类器，且分类器的质量直接影响生成效果。

#### 1.6.2 无分类器引导（Classifier-Free Guidance, CFG）

CFG通过在训练时同时训练条件模型和无条件模型（通过随机drop条件信息）来实现条件生成，无需外部分类器。采样时的预测噪声为：

\[
\hat{\epsilon}_\theta(x_t, t, y) = \epsilon_\theta(x_t, t, \varnothing) + w \cdot (\epsilon_\theta(x_t, t, y) - \epsilon_\theta(x_t, t, \varnothing))
\]

其中 \(w\) 为引导权重，\(w > 1\) 可增强条件效应。CFG已成为现代扩散模型的主流引导技术。

### 1.7 Score-based建模与概率流

#### 1.7.1 Score Function的定义

扩散模型的另一种统一视角是**Score-based建模**。定义**分数函数**（Score Function）为对数概率密度关于数据的梯度：

\[
s(x) = \nabla_x \log p(x)
\]

对于带噪声的数据分布 \(p_t(x_t)\)，相应的时间依赖分数函数为 \(\nabla_x \log p_t(x_t)\)。神经网络 \(s_\theta(x_t, t)\) 通过最小化Fisher散度来学习该分数函数：

\[
\mathbb{E}_{p_t(x_t)}\left[ \| s_\theta(x_t, t) - \nabla \log p_t(x_t) \|_2^2 \right]
\]

#### 1.7.2 随机微分方程（SDE）视角

Song等人统一了DDPM与Score Matching with Langevin Dynamics（SMLD），提出用随机微分方程描述扩散过程。前向扩散过程的SDE形式为：

\[
dY_t = f(t, Y_t) \, dt + g(t) \, dW_t
\]

其中 \(f(t, \cdot)\) 为漂移系数，\(g(t)\) 为扩散系数，\(W_t\) 为标准布朗运动。

#### 1.7.3 概率流ODE

前向SDE对应Fokker-Planck方程：

\[
\frac{\partial p_t(y)}{\partial t} = -\text{div}(p_t(y) \cdot v(t, y)), \quad v(t,y) = f(t,y) - \frac{g(t)^2}{2} \nabla \log p_t(y)
\]

由此可导出对应的概率流ODE（Probability Flow ODE）：

\[
\frac{dY_t}{dt} = v(t, Y_t), \quad Y_0 \sim p_0
\]

该ODE与原始SDE**在每一步具有相同的边缘分布**，提供了确定性采样的替代方案，可支持精确的似然计算和更快的采样。

**应用意义**：训练好的扩散模型可以同时支持：
- **SDE采样器**：随机版本，多样性更丰富
- **ODE采样器**：确定性版本，采样速度更快
- **精确似然估计**：通过ODE计算对数似然

## 第二章：强化学习中的扩散模型应用

### 2.1 模块边界与应用分类

强化学习与扩散模型的融合主要有三大应用类别：

| 类别 | 核心思想 | 代表工作 |
|------|---------|---------|
| **扩散规划器** | 在轨迹空间建模，生成完整状态-动作轨迹 | Diffuser、Decision Diffuser |
| **扩散策略** | 用扩散模型表示策略网络，生成动作分布 | Diffusion Policy、QVPO、LSEVPO |
| **数据增强** | 用扩散模型生成合成经验数据 | 经验数据扩增、分布内采样 |

### 2.2 扩散策略（Diffusion Policy）

#### 2.2.1 背景：为何需要扩散策略

在机器人操作等任务中，传统行为克隆（BC）将模仿学习视为监督学习问题，但其致命的**协变量偏移**问题导致策略在执行时产生累积性错误。尤其当演示数据中存在多模态行为（同一任务有多种合理解法）时，传统BC预测的平均动作往往是无效的。

扩散策略正是为了解决这一根本性难题而出现的。它将图像生成领域的扩散模型引入动作生成场景：从一个随机噪声向量出发，通过迭代去噪逐步生成高质量的动作序列。

#### 2.2.2 核心设计

扩散策略将**动作块（Action Chunk）** 作为扩散模型输出，即一次生成 H 步动作序列 \(a_{t:t+H-1}\)。执行时采用**滑动窗口策略**：执行前 K 步动作（通常 K=1），随后根据新的观测重新规划。

**为什么用动作块而非单步动作**：
1. **时序一致性**：动作块确保连续动作之间的平滑性
2. **推理效率**：块预测可减少在线推理的计算频率
3. **多模态能力**：块预测天然支持完整动作序列的多样性分布

#### 2.2.3 训练与采样

训练时，将专家轨迹中的状态-动作对作为条件输入，训练扩散模型学习条件去噪过程：

```
Algorithm 2: Diffusion Policy Training (BC)
Input: 专家演示数据集 D = {(s_t, a_t)_{t=1}^N}，时间步数 T
For each training iteration:
  1. 从数据集采样轨迹块 [s_t, a_t, ..., a_{t+H-1}]
  2. 采样时间步 k ~ Uniform(0, T-1)
  3. 向动作序列添加噪声: a_k = add_noise(a_0, k)
  4. 条件模型预测去噪后的动作: a_pred = εθ(a_k, s_t, k)
  5. 计算损失 L = MSE(a_pred, a_0)
  6. 反向传播更新 εθ

Algorithm 3: Diffusion Policy Inference
Input: 当前观测 s_t，扩散步数 T
  1. 随机采样初始噪声 a_T ~ N(0, I)
  2. For k = T-1 down to 0:
       a_k = denoise_step(εθ, a_{k+1}, s_t, k)
  3. 输出生成的 H 步动作序列 a_0
  4. 执行前 K 步动作（通常 K=1）
  5. 重复以上流程
```

#### 2.2.4 扩散策略的优点

扩散策略因其独特的概率生成范式，为机器人操作提供了三大实用优势：
- **多模态分布建模**：可以保留多种合理的动作方案，而不是退化为折中的平均动作
- **轨迹平滑性**：去噪迭代过程天然具有正则化效果，生成的轨迹平滑连贯
- **灵活的条件生成**：可以无缝融合图像、目标描述、语言嵌入等多模态条件信息

### 2.3 扩散模型在RL中的理论结合：优势加权回归

#### 2.3.1 理论基础

在约束型离线强化学习中，最优策略的解可以表示为：

\[
\pi^*(a|s) \propto \pi_\beta(a|s) \cdot \exp(\kappa \cdot A^\pi(s, a))
\]

其中 \(\pi_\beta\) 为行为策略（Behavior Policy），\(A^\pi(s,a)\) 为优势函数（Advantage Function），\(\kappa\) 为权衡系数。

扩散策略的训练目标天然支持这种优势加权形式：将优势函数作为**重要性权重**嵌入去噪损失中，使得高优势动作获得更大的训练权重。

#### 2.3.2 实现方式

在实际训练中，策略提升阶段将**非对称优势加权**后的反向扩散损失嵌入Actor损失中：

```
Algorithm 4: Advantage-Weighted Diffusion Policy Training
Input: 离线数据集 B，Critic网络 Qφ，Actor网络 εθ
# 策略评估阶段（更新Critic）
For each iteration:
  1. 从B采样(s, a, r, s')及下一状态的动作a' ~ πβ(·|s')
  2. 更新Qφ: minimize MSE(Qφ(s,a), r + γ·Qφ(s',a'))

# 策略提升阶段（更新Actor）
  3. 从B采样(s, a)
  4. 计算优势A = Qφ(s,a) - V(s)
  5. 将非对称优势权重融入去噪损失:
     权重 = softplus(A) if A > 0 else small_constant
  6. 更新εθ: minimize weight * MSE(噪声预测, 真实噪声)
```

### 2.4 扩散Q学习（Diffusion Q-learning）

#### 2.4.1 核心思想

传统Q学习中的策略通常是高斯策略，难以建模多模态动作分布。扩散模型凭借强大的分布建模能力，可被直接优化为Q学习中的策略网络——通过将去噪模型与 \(Q\) 值进行匹配来实现Q学习的优化目标。

#### 2.4.2 关键优势

- **天然探索性**：扩散策略不需要显式的熵正则或探索项即可保持多模态探索行为
- **避免模式崩塌**：相比GAN和VAE，扩散模型在多模态动作空间中的生成质量更优
- **约束自然满足**：扩散策略的迭代去噪过程隐式地对输出动作施加了平滑性约束

### 2.5 从零训练的RL扩散模型

传统的扩散模型RL方法通常是**在预训练扩散模型的基础上进行微调**——先通过最大似然估计学习一个基础模型，再用RL方法优化奖励函数。然而，Xuefeng Gao等人【16】提出了一种**直接从头训练**的RL扩散模型方法，无需任何预训练模型。

该方法的核心动机在于：当目标任务缺乏合适的预训练模型时（如新任务或特定任务），预训练驱动的微调方法面临模型误设定风险。他们的做法是将连续时间Score-based扩散模型的训练框架化为RL问题，利用连续时间控制或随机控制理论来直接优化奖励函数。

### 2.6 在线RL扩散微调

#### 2.6.1 LSEVPO：面向在线RL的扩散策略优化框架

扩散策略在在线RL中的整合仍是一个挑战。LogSumExp-weighted Variational Policy Optimization（LSEVPO）提出了一个兼具理论严谨性与实践效果的在线RL扩散优化框架。

LSEVPO的核心创新在于：
- **LogSumExp加权变分界**：提供对真实RL目标的更紧近似
- **自适应优势指数加权**：利用状态-动作优势值动态调整学习强度

该方法在MuJoCo基准测试上相比QVPO提升13.9%，相比PPO、TD3、SAC等传统基线提升27.6%。

#### 2.6.2 DSRL：潜空间RL驱动的扩散策略自适应

Wagenmaker等人【11】提出的Diffusion Steering via Reinforcement Learning（DSRL）采取了另一种创新路径：不在策略权重空间进行优化，而是在**扩散策略的潜噪声空间**运行RL，实现样本高效的策略在线自适应。DSRL仅需黑盒访问扩散策略，无需修改策略权重，并已在模拟环境和真实机器人任务中得到验证。

### 2.7 扩散模型作为规划器

#### 2.7.1 轨迹级扩散建模

与将扩散模型用作策略（Policy）不同，扩散规划器（Diffusion Planner）在**状态-动作联合空间**中进行扩散建模，生成完整轨迹。

**Diffuser**是这一方向的代表性工作，其采用**Classifier Guidance**利用奖励函数对生成的轨迹进行引导（steering），从而产生高奖励轨迹。具体而言，决策扩散模型先在低质量数据上训练，再用奖励函数引导其生成接近最优的轨迹。

#### 2.7.2 扩散规划与无模型RL的关系

扩散规划器更接近**基于模型的强化学习（MBRL）**——它学习环境的全局动态并进行规划。与之相对，扩散策略更接近**无模型强化学习（Model-free RL）**——直接学习从状态到动作的映射。两种范式各有适用场景：当需要全局一致性（如长时序任务）时，扩散规划器更优；当需要实时响应时，扩散策略更具优势。

### 2.8 扩散模型中的逆强化学习

#### 2.8.1 问题定义

逆强化学习（Inverse Reinforcement Learning）的目标是从观察到的专家行为中反推出潜在奖励函数。Nuti等人【14】提出了一种从扩散模型中提取奖励函数的方法，开创性地将扩散模型与IRL联系起来。

#### 2.8.2 核心方法

考虑两个决策扩散模型：一个建模低奖励行为的基线模型 \(p_\text{base}\)，一个建模高奖励行为的专家模型 \(p_\text{expert}\)。通过对比两者分布，可以提取出**相对奖励函数** \(R(\tau)\)，使得：

\[
p_\text{expert}(\tau) \propto p_\text{base}(\tau) \cdot \exp(\beta R(\tau))
\]

提取方法是将奖励函数参数化为神经网络 \(R_\phi(\tau)\)，对齐其梯度与两个扩散模型输出之差。

该方法的优势在于**无需环境交互**，仅需两个已训练好的扩散模型即可提取奖励函数。实验证明其在导航环境中能恢复正确奖励函数，且在标准运动控制基准上能有效提升基础模型的性能。该方法还成功泛化到图像生成领域，为有害图像分配较低的奖励。

## 第三章：总结与对比

### 3.1 扩散模型RL方法总览

| 方法 | 建模对象 | 学习范式 | 核心挑战 | 代表性工作 |
|------|---------|---------|---------|-----------|
| 扩散策略 | 动作分布 | 监督学习/离线RL | 协变量偏移 | Diffusion Policy |
| 扩散Q学习 | 策略+值函数 | 离线RL | 分布约束 | Diffusion-QL |
| 扩散规划器 | 轨迹分布 | MBRL | 规划效率 | Diffuser |
| 扩散逆强化学习 | 奖励函数 | 逆RL | 专家对比需求 | Nuti et al. |
| 在线扩散RL | 策略更新 | 在线RL | 安全约束 | LSEVPO, DSRL |

### 3.2 选型建议

1. **多模态动作场景（如机器人操作）** ：优先选择**扩散策略**——其天然的多模态建模能力可有效避免平均动作退化为无效动作的问题。

2. **长时序/复杂任务**：**扩散规划器**通过轨迹级建模可提供更强的全局一致性。

3. **环境可访问且需要自适应**：**在线RL微调方法**（LSEVPO/DSRL）可在安全约束下实现有效的策略自适应。

4. **仅有演示数据，需理解行为意图**：**扩散逆强化学习**可在无需环境交互的情况下提取可解释的奖励函数。

5. **快速部署可用模型**：基于BC的扩散策略是最直接的起点，其状态-Action映射可以直接从演示数据中学习，实现过程简洁高效。

### 3.3 前沿方向与展望

1. **扩散模型与基础模型的融合**：将Transformer架构（DiT）引入扩散模型，利用自注意力的全局建模能力与参数扩展性实现更强表达力

2. **实时性与效率优化**：通过ODE采样和蒸馏技术缩短推理延迟，使扩散策略可用于实时控制场景

3. **安全RL对齐**：利用扩散模型的概率特性设计安全约束，使在线学习过程保持在对训练分布的置信区域内

4. **具身智能与真实世界部署**：将扩散模型与多模态感知结合，构建端到端的具身智能决策系统
